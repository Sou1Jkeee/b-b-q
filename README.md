# Клон ask.fm на Ruby on Rails

У пользователей есть возможность задавать вопросы
другим пользователям, отвечать на вопросы заданные им и прочее.

## Установка и запуск

```
bundle install
rails db:migrate
```

### Демо версия

Демо версия доступна на хероку
