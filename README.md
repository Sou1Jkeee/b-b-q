# Приложение для организации мероприятий на Ruby on Rails

## Функции

- Создание мероприятий 
- Возможность создавать закрытые мероприятия (вход в мероприятие по пин-коду
- Добавление фотографий (Amazon Web Services)
- Почтовые рассылки

## Установка и запуск

```
bundle install
rails db:migrate
```
